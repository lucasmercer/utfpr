# Marco: Versão HTML Estática (Standalone)
**Data:** 02 de Maio de 2026
**Descrição:** Versão simplificada usando HTML5, Tailwind CSS via CDN e Lucide Icons. Esta versão foi criada para garantir compatibilidade total com serviços de hospedagem estática como GitHub Pages, evitando problemas de build ou caminhos base de frameworks.

## Características:
- **Tecnologia:** HTML Puro + Tailwind CDN
- **Ícones:** Lucide (via unpkg)
- **Design:** Polido, responsivo, modo escuro/claro integrado
- **Funcionalidade:** Link para DuoHora e QuiverVision mantidos

## Arquivos Principais desta Versão:
- `/index.html` (Contém todo o código da UI)
- `/favicon.png` (Ícone da página)
